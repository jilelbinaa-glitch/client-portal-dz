## خطوات النشر

### 1. Firebase Firestore — خطوة ضرورية
1. اذهب إلى [console.firebase.google.com](https://console.firebase.google.com)
2. **Firestore Database → Rules**
3. احذف المحتوى القديم والصق محتوى ملف `firestore.rules`
4. انقر **Publish**

### 2. رفع الملفات على GitHub
ارفع **كل الملفات** في المستودع (لا تفصل مجلدات)

### 3. تفعيل GitHub Pages
**Settings → Pages → Branch: main → / (root) → Save**
