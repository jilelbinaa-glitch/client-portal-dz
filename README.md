# بوابة العملاء — مكتب الدراسات المعمارية
## Client Portal - Architectural Office

بوابة عملاء احترافية وحديثة لمكتب معماري في الجزائر، تتضمن نموذج طلب متعدد الخطوات وتتبع الحالة والدردشة المباشرة.

### ✨ المميزات الرئيسية

#### 1. **نموذج طلب متعدد الخطوات (5 خطوات)**
- **الخطوة 1:** بيانات العميل الشخصية
- **الخطوة 2:** اختيار الخدمات ونوع المشروع
- **الخطوة 3:** تحديد موقع المشروع
- **الخطوة 4:** تفاصيل المشروع الإضافية
- **الخطوة 5:** مراجعة الطلب والإرسال

#### 2. **الخدمات المتاحة**
- 📐 الدراسة المعمارية (من 30,000 دج)
- 📋 ملف رخصة البناء (من 15,000 دج)
- 🔩 الدراسة الهيكلية (من 25,000 دج)
- 🏗️ الإشراف على الإنجاز
- 🛋️ التصميم الداخلي (من 20,000 دج)
- 💰 تقدير التكاليف (من 8,000 دج)

#### 3. **تتبع الحالة**
- البحث عن الطلبات برقم مرجعي فريد
- عرض حالة الطلب (قيد المعالجة، تم المراجعة، تم عرض السعر، إلخ)
- رسالة ترحيب تلقائية من المكتب

#### 4. **الدردشة المباشرة**
- التواصل المباشر مع فريق المكتب
- حفظ السجل الكامل للمحادثات
- تاريخ ووقت كل رسالة

#### 5. **التخزين المحلي**
- جميع البيانات تُحفظ في `localStorage`
- لا توجد حاجة لخادم خلفي
- البيانات تبقى محفوظة حتى بعد إغلاق المتصفح

### 🎨 التصميم

- **نظام ألوان فاخر:** ذهبي وأسود وكريمي
- **خطوط احترافية:** Playfair Display و Cairo
- **تصميم متجاوب:** يعمل على جميع الأجهزة
- **واجهة سهلة الاستخدام:** تجربة مستخدم سلسة

### 🔧 التخصيص

#### تغيير معلومات المكتب
عدّل القيم التالية في `index.html`:
- اسم المكتب: `مكتب الدراسات المعمارية`
- رقم الهاتف: `0550 000 000`
- البريد الإلكتروني: `info@office-archi.dz`
- الموقع: `الجزائر العاصمة`

#### تغيير الألوان
عدّل متغيرات CSS في `<style>`:
```css
:root {
  --gold: #b8923a;      /* اللون الذهبي الأساسي */
  --ink: #1a1714;       /* اللون الأسود */
  --warm: #f7f3ed;      /* اللون الدافئ */
  /* ... */
}
```
